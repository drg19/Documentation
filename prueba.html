Position


position: static → Se posiciona el elemento según el flujo normal de la pagina

relative → El elemento se posiciona según su posicion normal o inicial, el espacio dejado por el elemento al desplazarse se conserva

absolute → El elemento con posición absolute determina su posicion según el padre POSICIONADO mas cercano, si no ecnuentra un padre posicionado, se ubica según el body del documento

fixed → El objeto se posiciona según la ventana del navegador ysiempre permanece en el mismo lugar, al desplazarse el elemento,su espacio no es conservado

sticky → (sticky-info)


Los elementos usdos para posiionar un elemento son

left | right | top | bottom | z-index




display:inline →coloca a los elementos uno al lado del otro

display:inline-block →Fuerza a los elementos a ponerse uno al lado del otro,sin importar si son tipo bloque



float : left| right | none | inherit → Determina una posicion flotante dentro de la pagina para el elemento, posiciona todos los elementos con float apilados ya sea a la izquierda o derecha(Util para generar columnas dentro de la web) (Tambien se usa para rodear una imagen con texto, asignando el float a la img)


clear: → La propiedad especifica qué elementos pueden flotar junto al elemento borrado y en qué lado.


none- Permite elementos flotantes en ambos lados. Esto es por defecto

left - No se permiten elementos flotantes en el lado izquierdo

right No se permiten elementos flotantes en el lado derecho

both - No se permiten elementos flotantes en el lado izquierdo o derecho

inherit - El elemento hereda el valor claro de su padre clear:
